# How To Use

- Clone Repo: `git clone https://github.com/Inside4ndroid/Url-Resolver.git`
- Goto Directory: `cd Url-Resolver`
- Install: `npm install`
- Run: `npm run start` or `npm run dev`
- Open `http://your-ip-or-domain.com:3000?url=your-url-to-be-resolved`
